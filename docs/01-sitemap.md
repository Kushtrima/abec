# ABEC Fassadenbau — Sitemap

5-page corporate website (German, Switzerland market).

## Pages

```
/ (Startseite)
├─ Hero (video)
├─ About us teaser ("Gebaut auf Vision...")
├─ Why us (3 USPs with numbered list)
├─ Services teaser (3 cards)
├─ Featured projects (4)
├─ Process (3 steps)
├─ Baunetzwerk360 partner section
├─ Testimonials (3)
└─ Contact form

/ueber-uns (Über uns)
├─ Hero
├─ Unsere Geschichte (timeline: 2005, 2010, 2025)
├─ Unser Team (4 members)
├─ ABEC in Zahlen (4 stat cards: 20+, 500+, 100%, ZH/SZ)
└─ Warum ABEC (3 differentiator cards) + CTA

/angebot (Angebot / Services)
├─ Page hero
├─ 01 Design-Fassaden (Stein & Glas) — detail block
├─ 02 Sanierung & Energie — detail block
├─ 03 Verputz & Dämmung — detail block
└─ CTA banner

/referenzen (Referenzen / Projects)
├─ Page hero
├─ Project grid (up to 19 projects)
└─ CTA banner

/kontakt (Kontakt)
├─ Page hero
├─ Contact info (address, phone, email, hours)
├─ Contact form (Name, Email, Tel, Subject [dropdown], Message)
└─ Google Maps embed
```

## Global elements
- **Top nav:** Startseite · Über uns · Angebot · Referenzen · [Kontakt button]
- **Footer:** logo + tagline · menu · opening hours · contact info · copyright

## Languages
- Primary: **German (DE-CH)**
- Future: optional EN / FR (Swiss market consideration)
