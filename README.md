# ABEC Fassadenbau GmbH — Website

Project workspace for the ABEC corporate website (5-page German site).

## 📂 Structure

| Folder | Purpose |
|--------|---------|
| `assets/` | All raw media: images, videos, logos, fonts |
| `content/` | Editable copy, project data, testimonials |
| `docs/` | Sitemap, design system, content brief, analysis |
| `reference/` | Original client deliverables (PDF, PPTX) |
| `src/` | Source code (framework TBD) |
| `public/` | Static assets served by the framework |
